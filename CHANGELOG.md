## [0.1.0] — 2025-XX-XX

### Added
- Initial public release
- Cinematic scroll-driven landing experience across nine chapters
- Hero with parallax background, philosophy, signature coffee, culinary collection
- Interior walkthrough, crafted-daily, gallery, testimonials, reservation form
- Fullscreen concierge menu overlay with `localStorage`-persisted favorites
- Light/dark theme system with gold-accented palette
- Lenis smooth scroll, Framer Motion + GSAP choreography
- Custom cursor, canvas particle background, audio player, grain texture
- Mobile-responsive layout, reduced-motion support
- Vercel-ready configuration (`vercel.json`, optimized `next.config.ts`)
- MIT license, professional README, changelog
